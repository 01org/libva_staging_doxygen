struct {
   unsigned int   num_temporal_layers_minus1: 8
 The number of temporal layers with layer specific bit-rate targets that are supported. The application will send multiple VAEncMiscParameterRateControl and VAEncMiscParameterFrameRate structures for each layer, using the temporal_id field as the layer identifier. If per temporal layer rate control is not supported, num_temporal_layers_minus1 will be 0 and the temporal_id field in VAEncMiscParameterRateControl and VAEncMiscParameterFrameRate will be ignored.
