Detailed Description

Encoding region-of-interest (ROI).

The encoding ROI can be set through this structure, if the implementation supports ROI input. The ROI set through this structure is applicable only to the current frame. The number of supported ROIs can be queried through the VAConfigAttribEncRoi. The encoder will use the ROI information to adjust the QP values of the MB's that fall within the ROIs.
