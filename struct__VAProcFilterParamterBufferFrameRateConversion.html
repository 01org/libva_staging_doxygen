unsigned int _VAProcFilterParamterBufferFrameRateConversion::cyclic_counter

Counter within one complete FRC Cycle.

The counter would run from 0 to 4 for 24to60p in each cycle. The counter would run from 0 to 1 for 30to60p in each cycle.

unsigned int _VAProcFilterParamterBufferFrameRateConversion::num_output_frames

Number of output frames in addition to the first output frame.

If num_output_frames returned from pipeline query is 0, vaRenderPicture() will only produce one output frame with each call

VASurfaceID* _VAProcFilterParamterBufferFrameRateConversion::output_frames

Array to store output frames in addition to the first one.
